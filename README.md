# 🎮 坦克大战 (Tank Battle)

一款经典坦克大战（Battle City）风格的 HTML5 Canvas 网页游戏。

## 🕹️ 游戏玩法

- 控制你的绿色坦克消灭所有红色敌方坦克
- 保护地图底部的基地（🦅）不被摧毁
- 摧毁砖墙开辟道路，钢铁墙壁不可摧毁
- 每关消灭所有敌方坦克即可过关

## 🎯 操作方式

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` 或 `W` `A` `S` `D` | 移动坦克 |
| `空格键` | 发射子弹 |
| `P` | 暂停/继续 |

## ✨ 特性

- 🧱 可摧毁的砖墙与不可摧毁的钢铁墙壁
- 🤖 智能 AI 敌方坦克（追踪玩家、自主射击）
- 💥 粒子爆炸特效
- 🔊 简易音效反馈
- 📈 难度递增的关卡系统
- 💖 生命系统（3条命）
- 🛡️ 重生无敌时间
- 📊 得分与击杀统计

## 🚀 运行方式

直接在浏览器中打开 `index.html` 即可游玩！

```bash
# 或者用任意 HTTP 服务器
python -m http.server
# 然后访问 http://localhost:8000
```

## 🛠️ 技术栈

- 纯 HTML5 / CSS3 / JavaScript
- Canvas 2D 渲染
- Web Audio API 音效
- 无外部依赖
